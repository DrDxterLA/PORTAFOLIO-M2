# E-commerce Frontend

Este proyecto corresponde al frontend de una tienda e-commerce desarrollada
utilizando HTML5 y Bootstrap 5.

## Descripción
La tienda permite visualizar productos en formato de tarjetas, navegar entre
páginas, acceder al detalle de cada producto y simula un flujo básico de compra.

## Tecnologías utilizadas
- HTML5
- Bootstrap 5.3
- CSS3
- Git y GitHub
